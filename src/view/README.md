# ProjetCAOS
This is a README

Step by step :

1) Se placer dans le répertoire ("cd ...")

Pour uploader une modification :


2) git init
3) git remote add origin https://github.com/fmoins/ProjetCAOS.git
4) git add .
5) git commit --all
6) git pull origin master // création d'une nouvelle branche
7) git push origin master // fusion des branches

Pour récupérer les fichiers :
2) git init
3) git remote add origin https://github.com/fmoins/ProjetCAOS.git
6) git pull https://github.com/fmoins/ProjetCAOS.git
